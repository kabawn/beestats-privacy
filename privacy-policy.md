BeeStats is an early-access (beta) beekeeping management application designed to help beekeepers manage apiaries, hives, inspections, harvests, and related activities.

This Privacy Policy explains how information is collected, used, and protected when using the BeeStats application.

1. Information We Collect

BeeStats may collect or access the following types of information:

a) Personal Information

Email address, used for account creation, authentication, and user identification.

b) Application Data

Data entered by the user such as:

Apiaries

Hives

Inspections

Harvest records

Notes and observations

This data is created and managed directly by the user.

c) Location Data

Approximate or precise location, used to associate apiaries with a geographic area and improve application functionality.

Location access is used only when necessary and is not continuously tracked.

d) Camera Access

Camera access is used exclusively to scan QR codes for hive identification.

BeeStats does not take photos, record videos, or store camera images.

2. How We Use the Data

Collected data is used solely to:

Provide and operate the core features of the application

Allow users to manage and track their beekeeping activities

Enable QR code scanning and apiary localization features

Improve the application during the beta phase

Fix bugs and enhance user experience
